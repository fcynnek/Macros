The office you work for focuses on internal information communication technology (ICT) application development for the regional offices within United Nations.  You are responsible for the application testing and deployment of these applications.


Background Information

Recently a request to build an application which allows programme officers within the commission sessions to submit their meeting planning requests and progress updates. This application will be used by all five regional offices within the organization which are located in South America, Europe, Africa, Middle East and Asia Pacific. After the requests are approved by each programme director and then local conference management unit online, they will then be published on the corresponding regional offices’ websites. This is so that both internal and external meeting participants can search for these meetings and obtain needed information, and conference management units’ staff can also have an overview for all expected meetings.  Each type of person/user role would have different levels of access (some view only and some could submit request/edit request, and some would have approver role, etc.). The application has gone through internal module testing by the developers already and is now at a stage to involve users/super users from the regional commissions and conference units for User Acceptance Testing (UAT) for one month; it is then expected to be launched for all users one month after the testing is completed.
